Turbo-Plan is an AI-powered project management system designed specifically for software development teams. It integrates seamlessly with Claude Code through MCP (Model Context Protocol), enabling intelligent project tracking and automation.

## Key Features

- **Project Management** - Organize work into projects, issues, and milestones
- **Initiative Tracking** - Group related projects under strategic initiatives
- **MCP Integration** - Use Claude Code to interact with your projects directly
- **AI-Powered Agents** - Automated analysis and suggestions for issues
- **Work Queue** - Prioritized view of what to work on next

## Architecture

Turbo-Plan consists of three main components:

1. **Backend API** - FastAPI-based REST API for data management
2. **Frontend** - Next.js application for the web interface
3. **MCP Server** - Python-based MCP server for Claude Code integration
