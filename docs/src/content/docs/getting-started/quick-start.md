This guide will help you set up Turbo-Plan quickly using the hosted version.

## Prerequisites

- A GitHub or Google account for authentication
- Claude Code installed (for MCP integration)

## Step 1: Access Turbo-Plan

Visit [turbo-plan-web.fly.dev](https://turbo-plan-web.fly.dev) and sign in with your GitHub or Google account.

## Step 2: Create Your First Project

1. Click the **+** button or press `Cmd+K` / `Ctrl+K`
2. Select **Project**
3. Enter a name and optional description
4. Click **Create**

## Step 3: Add Issues

With your project open:

1. Click **Add Issue** or press `Cmd+I` / `Ctrl+I`
2. Enter a title and description
3. Set priority and status as needed
4. Click **Create**

## Step 4: Set Up MCP Integration

To use Turbo-Plan with Claude Code:

1. Navigate to the [MCP Integration](/guides/mcp-integration) guide
2. Follow the setup instructions
3. Start using natural language to manage your projects
